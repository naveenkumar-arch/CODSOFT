About Me

Hello! I'm an enthusiastic and self-motivated aspiring Full Stack Developer with a strong interest in software development, particularly in backend technologies like Core Java and MySQL, and frontend tools like HTML, CSS, and JavaScript. Apart from coding, I’m also improving my communication, teamwork, and problem-solving skills to prepare for technical interviews and professional work environments. My goal is to become a well-rounded developer who not only writes efficient code but also contributes positively to any team or project I work with.
